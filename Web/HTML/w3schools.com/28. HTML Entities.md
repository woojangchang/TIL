## HTML Entities

```html
&entity_name;
OR

&#entity_number;
```

- `<`나 `>` 등을 쓰려면 tag로 인식되어 쓰지 못하므로 `<`를 쓰기 위해서 `&lt;` 혹은 `&#60;`을 써야한다.
