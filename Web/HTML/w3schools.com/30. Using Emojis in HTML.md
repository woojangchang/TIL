## UTF-8 Characters

```html
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
</head>
<body>

<p>I will display A B C</p>
<p>I will display &#65; &#66; &#67;</p>

</body>
</html>
```

- A : `&#65`
- B : `&#66`
- C : `&#67`
